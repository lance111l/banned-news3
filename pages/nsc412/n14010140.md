<h4>
 非常国民大会纲领摘录
</h4>
<p>
 中华民国大陆沦陷区非常国民大会的纲领，由辛灏年拟稿，经非常国大筹备组修订，现已在网站公开发表，刊登网址为：
 <ok href="https://fcgd.cc/charter" rel="noopener" target="_blank">
  https://fcgd.cc/charter
 </ok>
 。
</p>
<p>
 <em>
  创立“非常国大”的基本目的
 </em>
 <br/>
 1、 团结各方力量，凝聚基本共识，支持中国大陆沦陷区人民的反共复国大业，确立“传播孙文思想、实现三民主义、推行去党留政、引导和平革命、光复中华民国、追求民主统一”的政治信念。
 <br/>
 2、 参与国际社会的舆论活动和政治活动，不失时机地宣传中国大陆社会对“光复中华民国，追求民主统一”的强烈政治要求，希望西方各国切勿再三违背中国民心，重走支持中国反动势力的错误历史道路，从此获得中国人民的谅解和感激，并以亚太区域稳定和世界和平等根本利益出发，终结共产主义全球化对人类的威胁。
 <br/>
 3、 诚邀台港澳和边疆三大民族各族各派海外代表参加非常国大，探讨周边地区民主转型各种方式，按和平民主平等之原则解决地区问题，避免流血冲突，并警惕中共离间和破坏民族关系，努力避免中共崩溃时，台湾新疆西藏和内蒙因被阴谋势力利用而分裂自立、中共残余势力乘机坐大而阻断大陆过渡期民主进程的危险。
 <br/>
 4、 培训未来中国民主法治人才，以中华民国自由区的成功现实及中共70余年暴政的真相，引导沦陷区人民推翻中共专制、重建中华民国、巩固沦陷区新生的民主宪政。
 <br/>
 5、 在中共崩溃之际，带领训练有成的自由区和海外“民国青年”，分赴大陆各省县，巡回宣传三民主义、协助民主选举，宣扬中国大陆主权与治权回归中华民国，号召军队国家化、各级机关去党留政，并采取“先上下，后中间”之选举策略，首先由下而上，推行县级自治，巩固民主基础，建立县级临时议会，选举临时政府和各县国大代表，迅速组建全国性的国民大会，真正实现在第一时间将权力还给人民，这样才能从根本上杜绝各省区分裂与割据、及乘乱建立军事独裁统治的任何可能，防止保共改良派勾结中共军头和官僚，企图由上而下地擅自建立各级临时政府，重蹈武昌起义后保清改良派勾结旧军人旧官僚迅速劫夺和瓜分权力、擅立各省军事独裁政权、直至谋图复辟之覆辙，同时也吸取苏共崩溃后普京化之历史教训。
</p>
<p>
 <em>
  “非常国大”的政治性质
 </em>
 <br/>
 1、 在海外创立“中华民国沦陷区非常国民大会”（简称非常国大），是由于大陆沦陷区虽为中华民国之固有疆域，但沦陷区人民政治权利被中共专制统治剥夺了七十余年，沦陷区人民完全没有合法的民意代表机关，因此，沦陷区人民根据1946年中华民国宪法，乃有权在沦陷区外任何地方，包括中华民国自由区，自行建立民意代表机关。
 <br/>
 2、 非常国大是在非常时期，用非常方式为沦陷区人民所设立的境外代议形式，因而是议事机构，不属于、也不是任何政党和政治团体。其所有议事内容，以推动“反共复国”为共同目标。因为只要实现了反共复国，民国沦陷区就能够走向民主。
 <br/>
 3、 非常国大不是大陆沦陷区人民的“民选议会”，故不具有人民所赋予的政治权力。其一切行为，均在推动和引导；其一切成果，只能奉献给沦陷区人民自行选择或参考。
 <br/>
 4、 鉴于长期以来境外民主政治组织为中共严重渗透和破坏，非常国大不由选举产生，而由境内外有志于民主复国的爱国人士协商组成，同时也不在境外办理任何公开竞选事宜。◇
</p>
