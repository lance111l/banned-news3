<div><p>
 【大纪元2023年06月13日讯】
 <span class="s1">
  （大纪元记者施萍报导）
 </span>
 <span class="s2">
  日前，在一次美国东北英格兰地区的社区活动上，一位麻省的
  <ok href="https://www.epochtimes.com/gb/tag/%E8%89%BA%E6%9C%AF%E5%AE%B6.html">
   艺术家
  </ok>
  得知了纽约州新创科技公司“
  <ok href="https://www.epochtimes.com/gb/tag/%E5%B9%B2%E5%87%80%E4%B8%96%E7%95%8C.html">
   干净世界
  </ok>
  ”的消息，她显得很兴奋。
 </span>
</p>
<p class="p5">
 <span class="s3">
  “我认为我们这个世界应该来了解‘
  <ok href="https://www.epochtimes.com/gb/tag/%E5%B9%B2%E5%87%80%E4%B8%96%E7%95%8C.html">
   干净世界
  </ok>
  ’，这很重要。”莎莉‧切特温德（Sally Chetwynd）说，“这是净化社交媒体的第一步。”
 </span>
</p>
<p class="p5">
 <span class="s3">
  切特温德是一位文案作者、诗人、小说家、文字编辑、历史学者、画家和音乐人，她是麻省威克菲尔德Brass Castle Arts艺术工作室的主人。
 </span>
</p>
<p class="p5">
 <span class="s3">
  切特温德是老派美国人的一员，她从父母身上学到了“创造力、节俭、独立和务实”等传统的“美国精神”。
 </span>
</p>
<p class="p5">
 <span class="s3">
  她的丈夫是历史重演剧中扮演美国第16任总统林肯的专业演员，她也称自己是“活着的历史学家”。
 </span>
</p>
<p class="p5">
 <span class="s3">
  切特温德认为，“现在社交媒体上有太多的邪恶。”
 </span>
</p>
<p class="p5">
 <span class="s3">
  “什么女孩变男孩，男孩变女孩，或者全面否定你的人性——他们来自那里？来自邪恶，现在有很多的邪恶。”她说，人们在这样的环境下不知如何是好，“我认为我们的世界充斥着邪恶的东西，它渗透进所有的事情，体现在我们生活的每一个方面，让人防不胜防。”
 </span>
</p>
<p class="p5">
 <span class="s3">
  在这样的情况下，人们只能闭上眼睛、关上耳朵，躲进自己的角落中，“我们说：我们什么也改变不了……”
 </span>
</p>
<p class="p5">
 <span class="s3">
  就在此时，切特温德在社区活动上听到了“干净世界”的消息，一个立志为家庭、家长和孩子们提供“干净”的社交媒体的消息。这让她兴奋与感到安慰，更重要的，“干净世界”的概念给她以启发。
 </span>
</p>
<p class="p5">
 <span class="s3">
  切特温德回想起多年前一位同事母亲的一段往事。当时这位女士住在费城城郊的一个破旧的地方。房子没有粉刷，街道上推满了破汽车和垃圾。有一天，有人在房子的台阶上摆上了一盆天竺葵。几天之后，另外一个人在自己的台阶上也放了一盆鲜花；再后来，有人清理了垃圾，修剪了草坪，种上了越来越多的鲜花。最后，那里成了一个让邻里们骄傲的社区。
 </span>
</p>
<p class="p5">
 <span class="s3">
  “干净世界”让切特温德受到了同样的鼓励，就像那盆天竺葵，让她看到了社区美好的明天。
 </span>
</p>
<p class="p5">
 <span class="s3">
  “是，我们不能改变这个世界，但是我可以改变这个世界的一个小角落，清理一下这个小角落，这是我能做到的，这也可以鼓励其他人；就像你把小石头扔进池塘里，会引起涟漪，然后涟漪会传到整个池塘，触及到一切。”
 </span>
</p>
<p class="p5">
 <span class="s3">
  切特温德说，“这是我们可以一步一步扭转局面的事情。因此，‘干净世界’就是洁净社交媒体的第一步，他们提供一个有益于家庭的材料和选择，这是一个将您的信息传播到世界的地方；是一个为我们的孩子提供良好的、清洁的娱乐和教育内容，让他们健康成长的地方。”
 </span>
</p>
<p class="p5">
 <span class="s3">
  切特温德的理由是，这是因为“干净世界”是人们发自心中的“爱”的产品。
 </span>
</p>
<p class="p5">
 <span class="s3">
  “从‘爱’我们获得‘希望’，而‘希望’可以让我们改变一切。”她说。
 </span>
</p>
<p class="p5">
 责任编辑：孙芸#
</p>
