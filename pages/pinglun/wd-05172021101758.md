 <p>
  最近几年，国际政治的焦点逐渐从欧洲和中东转移到印太地区。中国在南海和台海采取的扩张和军事行动，提升了亚太各国的警惕心。以澳洲、日本、印度为首的亚太地区主要国家，已经开始进行全方位的合作与结盟；欧洲在美国的敦促下，也开始派军舰进入亚太地区，展现西方国家的团结。但中共面对区域国家和国际社会的反弹不仅没有收敛，而且自认为国力已经强大到足以与美国为首的同盟国对抗，备战的动作和意愿也日趋明显。
  <br/>
  <br/>
  针对这样的局势发展， 亚太国家到底应当做什么，以应对严峻的形势呢？澳洲高级情报官员罗里·梅尔卡夫在他的新作《亚太竞逐》中提出了四点主张，我认为是相当中肯而实际，值得外界重视的。
  <br/>
  <br/>
  首先，作者认为，准备应付局势恶化的一个重点就是未雨绸缪，而未雨绸缪的重点在于“培养能力”。日本和澳洲现在开始帮助训练其他国家的海岸防卫部队，并且提供船只供海上监测与通讯设备给伙伴国家。包括从越南到印尼，从索罗门群岛到斯里兰卡的多个国家，就是这样的区域性合作以增强各国抵御能力的范例；
  <br/>
  <br/>
  其次，作者认为，不管日本、印度和澳洲如何已经觉醒并开始备战，但美国的支持还是至关重要的因素。他建议，美国和所有支持军力平衡的国家都需要在传统军事力量，以及未来的科技和地缘经济的韧性上取得优势，就是抵抗压力以及施加压力的能力。它们需要恢复两种形式的威慑力量：不让敌方获利，以及还击的能力。这将牵涉到维持军队现代化，投资新的作战工具，像是网络、人工智慧、太空、新材料、机器人和先进运算等。随着中国不断提升自己的军事实力，美国也必须在这方面加强投资。
  <br/>
  <br/>
  第三，作者呼吁亚太地区各国之间需要经常举行对话，并以坦诚的精神交换意见，取得共识。这样的对话，应当包括政治层级的对话。作者认为，对话不能只停留在一年一次或两次的会晤上，各国之间在对话之后，应当成立具体负责执行的工作小组，提供像情报与后勤这些问题的资讯给各国分享。针对危机情节，各国也需要提前进行讨论和进行作战演习。
  <br/>
  <br/>
  最后一点，我觉得最值得外界深思，也是作者提出的最重要的论点之一：在他看来，要制衡中国势力，印太国家必须愿意付出代价，忍受短时期的痛苦。这需要领导人的政治意志力。这一点非常重要，是因为国际政治，有的时候不仅仅是靠国家实力，有的时候比的也是领导人的政治意志。同样在印太地区，面对中国的攻势，澳洲和新西兰的政策就大有不同。澳洲面对中共的经济制裁宁愿付出一定代价，也不放弃对于自己国家利益和价值观的坚持；而新西兰则显然为得到中国的经济红利，而放弃了一些原则。因此，印太各国政治领袖的态度也很重要。
  <br/>
  <br/>
  最后，作者在书中讲述了一个小故事，我觉得非常耐人寻味：当越南和中国海防部队船只在靠越南海岸线的南海海域中，为了争夺石油和天然气探勘权而相互推挤、冲撞时，一位越南外交官在与外国学者开会被问到一连串问题：如果他的国家明天和中国发生战争了，谁会前来援助呢？“这个答案很简单”，这位外交官不慌不忙地答道：“越南人”。我想，这才是亚太地区国家自我防卫的最好心态。
  <br/>
