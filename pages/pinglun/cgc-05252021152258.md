 <p>
  距离1989年那场既波及全国又轰动全球的民主运动已经32周年了，中共沦陷区人民至今依然挣扎于共产专制政权的铁蹄之下。“六四”屠杀过去32年，不幸的是中共刽子手们仍然没有被追责清算。在此情况下，这片土地上怎么会出现正常社会人心所向的那种公平正义？！也就是说，从1989年“枪声一响，由偷变抢”开始，中共沦陷区社会从未停止向深渊滑动，谈不到朝正义的方向前进半步，更何谈社会实现基本的公平正义、人们持守道德规范！
  <br/>
  <br/>
  中共沦陷区社会要想走向“以邪正善恶论是非曲直”，“以是否合情合理，是否具有正当性”等基本常识作为衡量事物的标准规范，必须要等到实现……，或者开始实现转型正义的时候，才能从根本上真正开始。否则就像是打不好基础的建筑，不会有长久的发展前途。 而要实现转型正义，就必须从源头进行。不从根本上进行的“转型”，不单单不彻底，而且必然留下后患，社会早晚仍会走入死胡同，最终导致无法前行。然而，要在中共沦陷区推行转型正义就无法绕开必须终结中共暴政这一现实问题。在这个不得不面对的问题上，只有“怎么做”的问题，而没有“该不该”，或是“要不要做”的问题。
  <br/>
  <br/>
  如果要破除专制体制，国际上的支持非常重要，必不可少，甚至起着决定性作用。但是必须设法解决，如何跳出“政治正确”—— “和理非”范围中，解决问题的束缚。这的确很不容易做到。然而正因如此，中华儿女们才更需要开拓思维、拯救涣散，自己主动组织起来，从破除中共在国内的暴力维稳出发采取行动、夺回主动权，使得国际社会面对现实，认可人民反击暴政是符合国际法“正当防卫”的基本权利。
  <br/>
  <br/>
  纵观全球，只要以美国为人权盟主的民主国家还停留在想要通过“与中共政权合作解决问题”的层面上，就不存在口头之外的，对中国人民的真正支持。也就是说，只有当各民主国家内，各不同政党多数都从内心真正认识到，必须把中共暴政扫进历史的灰烬世人才能安全的时候，时机就成熟了。其实只要使得他们改变隔岸观火的心态，明白消灭人类的公敌不只是在帮助别人，更是在拯救自己，足矣。
  <br/>
  <br/>
  而要推动这一天的早日到来，我们身居海外者必须首先打破“政治正确”的束缚，把战略融入政治诉求当中，使得西方的政客们无法回避中共对内迫害本国人民、对外破坏国际文明秩序，威胁西方国家人民的自由生活这一事实。
  <br/>
  <br/>
  实践证明，影响人民远比影响政客要容易得多；通过专家（比如苏联专家）影响政客，要比通过普通百姓有效得多。当西方国家的人民要求“政客们必须做正确的事情”的声音足够高时，一切也就水到渠成了。当然，要实现这一切需要有人专门有计划、有步骤地持续运作，而且不要期望一蹴而就、立竿见影，而是前期只问耕耘，相信春种秋收，作好正确定向后，艰苦努力的思想准备。
  <br/>
  <br/>
  每年纪念“六四”，悼念“六四”屠杀的遇难者，会让世人勿忘八九“六四”历史真相，永记中共的罪恶。然而这还不够，人民必须探索出更进一步的方法，让用行动铲奸除恶成为全球正义之士联手行动的共同目标。相信只要你我和一切受共产暴政之害的各国有正义感的人们群策群力，不仅仅停留在坐而论道，而是起而行之，就一定能实现铲除暴政的目标。
