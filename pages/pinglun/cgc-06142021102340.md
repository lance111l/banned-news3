  很可能是因为源于撒旦教的那个组织在中国成立的纪念日快要到了，中共加紧了在全国范围内，对觉醒者进行打压的力度。最近，无论是“喝茶”还是随意绑架合法公民的频率都加快了很多。
  <br/>
  <br/>
  2021年6月12日下午三点左右，“709”律师谢燕益在吉林四平华谊新城郭洪伟父母的廉租房处，看受害人狱中死亡的郭洪伟案件材料时，被四平公安破门而入绑架。郭洪伟的相关案件证据材料被抢劫一空，然后谢燕益律师被带往铁东分局平南派出所。谢律师能记住的现场参与人员有警号302336的王悦（音），警号302115，现场指挥负责人钟局长。押解谢律师回京的四位劫匪穿着便衣，抵达北京后接应的吉林七座警务车牌号是“吉C0088”。
  <br/>
  <br/>
  按照中国的宪法和法律，无论是阻止律师办案、实施绑架，还是破坏案卷、毁灭证据，都是犯罪行为；即使司法部门也不能例外，必须按照《刑事诉讼法》的规定程序行事，否则就属于违法。

  <br/>
  <br/>
  从绑架后在北京交接时的不断请示，到最后中共做出把谢燕益扔在派出所门口这样荒唐的处理结果，至少说明：第一，中央、地方无缝对接实行联手绑架，绝非单纯是地方政府的乱权妄为，而是中共刻意打造的“维稳”系统的一贯做法。第二，中共动用土匪手段非法阻止律师办案，不但明显违宪违法，还使中共具有了参与杀害郭洪伟的重大嫌疑。第三，也是最重要的、起决定性作用的一点：不是中共不够无耻，也不是中共不想用“口袋罪”，而是在709大抓捕中“抓了一个，站起来一家”的教训实在是太深刻。那次709全国性大抓捕，不仅没能震慑住觉醒者，还使得党国的遮羞布彻底被撕掉，使中共在国际上颜面丢尽，始作俑者那种感觉实在不好受。为了避免再次触发如709妻子们的群起反抗，中共权衡利弊后，只好把谢律师给放了。
  <br/>
  <br/>
  其实，几十年来，中共从来就没有停止过用土匪绑架的手段迫害公民。每当理屈辞穷又不得不掩盖它的罪行时，中共就会彻底脱掉伪装的羊皮，露出强盗的本性。2005年，中共当局在北京公然绑架郭飞雄和我；2006年，在山东绑架正在声援我的高智晟律师及前来声援我的朋友们；把我妈妈和我的孩子从北京绑架回山东；2007年，绑架我的妻子袁伟静，不让她去北京；同年，李和平律师被绑架殴打后扔在路边；2008年，绑架殴打艾未未；2009年，绑架殴打张凯、李春富；邓玉娇案件中，威胁《新京报》女记者孔璞、《南方都市报》记者等；2010年，绑架酷刑范亚峰、余杰；绑架钱云会案件的目击证人钱成宇；2001年“茉莉花革命”期间，绑架酷刑江天勇、李和平、刘晓原、艾未未、黎雄兵、李天天、唐荆陵、滕彪等等律师；特别是，因为探访东师古而被中共绑架殴打的人员不计其数；还有2015年，绑架吴淦——网名屠夫、江西新馀的刘萍，以及后来的709律师们和声援709律师的律师妻子们；2017年，再次绑架高智晟律师，至今三年多了，如同“人间蒸发”般的生死不明；还有疫情以来，被绑架的方斌、陈秋实、张展等等……，中共的罪恶罄竹难书。
