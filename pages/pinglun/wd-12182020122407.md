 <p>
  各位听众你们好，我是王丹。在美国大选基本上已经尘埃落定之后，中国的外交也开始转守为攻，其中一项重要的行动，就是对澳洲采取了积极进攻的态势。中澳之间，这几年本来就冲突不断，但中国比较处于守势，但最近则不断透过贸易手段，对澳洲进行制裁，从红酒，牛肉，到棉花，大麦，中国对澳洲的产品在关税等方面的封杀行为越来越多。10月开始还限制澳洲煤炭的进口，导致几十艘运煤船只滞留在海上。
  <br/>
  <br/>
  中国明知道对澳洲的制裁，尤其是对煤炭的限制，同时也会令本国的生产受到冲击，但仍然这样做，当然是有明确的战略目标的。打击澳洲，可以说是警告，也是试探，针对的主要还不是美国，而是美国的盟友，尤其是欧洲国家。因为澳洲和欧洲国家一样，在经济上已经对中国有了一定的依存度。中国要通过对澳洲的制裁，测试两点：第一，西方国家摆脱经济上对中国的依赖的决心有多大，可行性有多高？第二，西方国家在面对中国的挑战的时候，是否能够团结一致。
 </p>
 <p>
  <figure class="image-richtext image-inline captioned" style="width:620px;">

 <p>
  <br/>
  <br/>
  其实，第一条在某种程度上，也取决于第二条。因此，在这次中澳大战中，西方国家作为一个阵营，要做出什么样的反应，就是一个重要的考验。毕竟，如果澳洲商品出口中国受阻，其它一些西方国家完全可以成为两虎相争之下的受益者，他们的出口商可以取代澳洲大捞一笔。同盟的集体合作，与本国的经济利益之间，有些国家必须要做出选择。这个选择，不仅决定着澳洲是否可以抵挡得住中国的经济攻势，也会成为中南海的决策者据以制订未来的外交政策的参考。
  <br/>
  <br/>
  目前看，澳洲正在得到西方盟国的支持。前几天针对中国对澳洲红酒征收巨额反倾销税，来自新西兰，德国，英国，意大利，美国，挪威，日本，丹麦等多国的议员，透过影片发起了全球声援行动。发起行动的跨国议员联盟来自18 个国家及欧洲议会，有170多名各国跨党派议员组成，虽然其声援只是道义立场的表达，但对于澳洲来说，也可以感受到温暖。据澳洲媒体报道，“五眼联盟”正悄悄商量一项计划，准备联合起来针对中国产品实施“报复性制裁”，以反击中国对澳州的贸易制裁。他们考虑的另一个方案是，澳州对来自中国的产品征收报复性关税，如果中国试图从四个同盟国弥补损失，美国，英国，加拿大和新西兰将拒绝从中国购买相关产品。这样的计划一旦施行，就是比道义声援更直接的支持，对中国来说，将是强有力的反击。
  <br/>
  <br/>
  不过我们也要看到，澳洲虽然得到西方盟国的支持，但是在亚洲地区，美国和日本的影响尚未能导致对中国的制约。虽然日本对澳洲的支持态度很明显，11月17日澳洲总理访问日本，双方针对签署“相互准入协定”达成了共识，建立了“准同盟”关系，但是，包括缅甸，柬埔寨，老挝，尤其是战略地位重要的泰国和新加坡在内的一些国家，在中澳之争的问题上，还是采取观望态度，不愿意得罪中国。显然，美国要支持澳洲，必须加强印太战略的布局，对亚洲一些“墙头草”国家施加更大的压力，才可能遏制中国对澳洲的攻势。
  <br/>
  <br/>
  中国突然对澳洲采取强硬立场，有其杀鸡儆猴的战略意图。如果西方国家不能团结一致，力挺澳洲，未来很可能会面对同样的经济攻势而被一一攻破，因此，这场中澳大战，已经不仅关系到澳洲一个国家，而是对整个西方社会的一个考验。以上是自由亚洲电台特约评论员王丹为您做的评论。
 </p>
