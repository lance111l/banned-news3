<div id="storytext">
 <p>
  最近因为前中共总理温家宝的文章在党的喉舌上，被中共下架产生了一系列连锁反应。在中共党内产生的争议或者分歧究竟有多大尚且不十分清楚，可是在民间有关温家宝到底是良相还是影帝的争论，高潮却是一波超过一波。而且不仅在中共沦陷区的网上网下如此，就连在海外的华人各方，同样是为此各抒己见，甚至唇枪舌剑互不相让。看上去似乎令人眼花缭乱，一时难以定论。
  <br/>
  <br/>
  反观历史，自从温家宝2002年出任中共总理十年间，他确实在不同的场合说了一些不仅听起来令人很舒服，并且给人充满期待的话。特别“高大上”的比如：“没有政治体制改革的成功，经济体制改革不可能进行到底，已经取得的改革和建设成果还有可能得而复失，社会上新产生的问题也不可能从根本上得到解决，文化大革命这样的历史悲剧还有可能重新发生”，“公平正义的光辉应该就像太阳一样”等等。在具体事情上，无论是汶川地震时说“一定要追查豆腐渣工程中的腐败，追究责任人”，还是浙江温州动车出轨事故中说“要彻查事故原因”……，温家宝的很多“豪言壮语”都令人觉得无需采取什么行动，只要相信党中央、等着中共中央展开调查就行了。如果你真的相信诸如“中央还能骗人吗？”之类的话语，有可能成为习惯受骗的“巨婴”。
  <br/>
  <br/>
  温家宝说“一定要把事实查个水落石出，还历史一个公道，给人民一个交代……”，听起来是多么高大上！令人无懈可击！可是，我们必须要问一句：哪一项他真正用行动去践行了？！更别说是实现了的！众所周知，以前就连温家宝自己也常说：“听其言还要观其行”，老百姓也常说“光说不练假把式”。有谁能举出一个可信的、能够查证的实例来证明，温家宝曾经为了去实现他自己的诺言而就哪件事情努力尝试过？只是因为权力太小，有心无力最后没能成功的例子吗？恐怕没有人能够举得出来，因为这样的事情就没发生过。
  <br/>
  <br/>
  尽管温家宝自己在这次写的（后被中共下架的）文章中说：“我同情穷人，同情弱者，反对欺侮和压迫。我心目中的中国应该是一个充满公平正义的国家，那里永远有对人心、人道和人的本质的尊重，永远有青春，自由，奋斗的气质。我为此呐喊过，奋斗过。这是生活让我懂得的真理，也是妈妈给予的。”也许温家宝曾经真的希望这样，也许他在上世纪八十年代真的为此奋斗过，不过“六四”屠杀发生以后，如果那些我们都知道的、他说过的“高大上”语言就算是他为此呐喊过，我们姑且认可，但是我不认为他真的为此努力奋斗过。
  <br/>
  <br/>
  别的我不知道，2012年4月，当时我为了躲避中共的第三次黑帮绑架而进入美国使馆避险时，据亲自参与同中共谈判的美国官员回来说：“这件事情已经到达了中共中央最高层，胡锦涛和温家宝都知道了。现在就是个面子问题……。”
  <br/>
  <br/>
  确实如此，“同情穷人，同情弱者”这样的说法都对，作为一个仁者，确实应该具有悲天悯人的情怀，同时更应该具有“傲上而不忍下，欺强而不凌弱”的英雄气概与担当。可是从我2005年被中共绑架回山东东师古软禁、殴打，到后来被构陷判刑入狱关押，一直到坐满最后一天牢狱，又立即被中共挟持回家继续遭受更严格的非法拘禁，这一切对我们的打压迫害历时近七年，超过胡温当政十年的三分之二以上的时间。据美国前总统布什先生亲口对我说，他曾经直接向胡温提出过我的事情，当时还是雷德做驻华大使。因此胡锦涛和温家宝从一开始就知道我被迫害的事情，绝非是我进入美国大使馆之后才知道。当时的使馆官员也只是估计着说：“中共党内，像温家宝这样的改革派官员是支持你的。”见我露出不解的表情，他又补充说：“隔着桌子从他们的表情就能看得出来……。”可是，有谁听说过温家宝对此事表过什么态吗？
  <br/>
  <br/>
  当然，“水至清则无鱼，人至察则无徒”，对于真正认清共产专制体制邪恶本性的任何人，不管是体制内还是体制外的，只要他确实用实际行动想要推翻共产专制体制，决心与邪恶中共决裂，就是同道中人，必须共同联手进行反共行动，他们当然有权参与中国的革命行动。就算有些中共干部曾经参与对人民的迫害，只要他愿意弃暗投明，立功赎罪，在转型正义的过程中也应该根据具体的实际情况进行将功补过。何况即使是中共干部，也只是清算少数手上沾着人民鲜血者，多数党员进入体制只是想利用垄断一切的专制体制最大限度地攫取利益，对那些看到中共大势已去再行动的投机者则另当别论。
  <br/>
  <br/>
  不过，仅仅因为像党的温总理这样的高官，只是说了几句和中共魔鬼不太一样的话就做过度的解读，很容易误入歧途。比如这次温先生的文章，人家最多算是党内老领导对后辈做的某件事情表示不同看法而已，就连批评都算不上，就解读成为“不满”；文章中连宪政法治、新闻自由、民主制衡都没有提到，就将其解读为“有可能与我们站在一起”，实在脱离事实太远。你怎么知道后辈做的其他事情他们没有一起商量决定？至于把这个看成是“党内有人要站出来反共，我们应该团结，联合，鼓励他们，否则其他体制内的人就不敢站出来反共了”的解读，根本就是无的放矢，实在是一厢情愿，解读发挥得太过离谱！
  <br/>
  <br/>
  在目前情况下，对于体制内的中共干部而言，如果站出来反共只有流亡和进监狱两个结果，谁都不会仅凭着良知就放弃既得利益，主动起来反共。除非是因为在体制内失势、危险正在降临，不得已而为之。除了一些信仰者，又有谁是良知使然呢？反之，如果人民的力量足够强大，势不可挡，中共拒绝变革将会死得更惨时，党内主动弃暗投明者会大有人在。
  <br/>
  <br/>
  因此，即使温家宝再怎么同情穷人、同情弱者，在官民矛盾如同水火，根本无法调和只能靠暴力维稳的今天，试问他肯为了穷人与弱者的利益去以失去自己利益冒险，不顾中共的利益站在人民一边吗？如果不是，对“他们会反共”的解读不是无的放矢又是什么？如果说，九个常委各管一摊别人不好插手，那么就连同事都不愿得罪的人会愿意得罪整个党吗？
  <br/>
  <br/>
  另外，不管他的初衷是不是帮着中共唱白脸只说不做，给人民虚假希望的效果就是起着帮助中共维稳的作用。作为一个在中南海里工作近三十年、十年担任总理的人来说，毫无疑问是深知“明知不可为而为之”的后果的。因此，他绝不会为了他口中的“公平正义”或者“同情弱者”，而做出什么自毁前程的事情来。这一点从，他们掌握权力的时代都早已成为过去式，“好事没办成，坏事却做了不少”，就足以证明了。
  <br/>
  <br/>
  在2008年5·12 大地震中，人家为了避免灾民群情激愤形成失控，说几句诸如“一定要追查豆腐渣工程中的腐败，追究责任人”等等安抚、麻痹四川灾民，以解燃眉之急，许以空头承诺，很多人至今不仅没有意识到上当受骗——中了共产党的缓兵之计、错过了凝聚民心的战机，还信以为真了。接下来，进京上访要求兑现承诺“调查豆腐渣工程中的腐败问题”，遭到大小流氓上下联动、绑架截访成为维稳对象，才是“骨感”的现实。
  <br/>
  <br/>
  综上所述，中国的未来唯有抗争，不拘一格的抗争。在“党为刀俎，民为鱼肉”的情况下，人民只是他们眼中的“韭菜”。小羊拿什么和狼合作、联手？因此，对于统治者，人民不仅要听其言，更要观其行。尽管一些漂亮的话说了比不说要好，但是人家只是有“战略策略”地说说而已，就被尊称为“良相”。人家中共会不会觉得，你们被卖了还在帮着他们数钱？至于在胡温时期，对打压所有要求针对四川地震中的豆腐渣工程展开调查、追究责任的人，如黄琦、谭作人等被构陷判刑；对高智晟和我等维权者被中共视为敌人的人的统一打压迫害行动中，他是否表示过反对？我想答案是否定的。那他就对中共这些邪恶行为同样负有责任，难脱干系。
  <br/>
  <br/>
  人民在组织起来形成排山倒海之势之前，一定要保持清醒，必须知道谁才是我们人民的公敌。即使不能从政治和军事的战略高度找到砸烂共产专制体制的良策，至少不应该被中共干部中演得太真的政治演员带进戏里。尤其是我们这些曾经遭受过中共邪恶政权直接迫害的人们，终日打狼难道还不了解狼的本性吗？！
 </p>
