 <p>
  随着中共统治下权贵资本主义体制的成长，阶层固化趋向稳定，资源配置丧失了流动性和灵活性。各领域、各层级都呈现内耗性的恶性竞争，被称为社会内卷化，“内卷”一词由此广泛流行。总加速师的加速主义对投入内卷竞争的积极态度赞赏有加，号召：撸起袖子加油干！然而，历经内卷恶斗的人们最后发现，没有赢家，都是输家。
  <br/>
  <br/>
  于是在百年建党大庆临近，习大帝准备20大继任的时刻，厌倦内卷、离开内卷的情绪和意识，突然以一个无奈卑微的“躺平主义”表现出来。“躺平”，顾名思义，懒散躺着，无欲无求，不争不抢，远离内卷。
  <br/>
  <br/>
  有分析人士说：“躺平主义”在中国兴起，主要是因为80后、90后的青年一代在包括买房、租房、结婚、生子、买车以及在职场上，面临巨大的生活压力。其深层次原因在于，中国的社会制度导致中国劳工长期低工资、低福利和低人权。
  <br/>
  <br/>
  “躺平主义”的内涵不断增生，被解读为是大众自发的无产阶级运动，抵抗资本剥削，表现为消费降级，最低限度社交。“不买房、不买车、不结婚、不生娃、不消费”，六不主义，花最低的劳动“维持最低生存标准，拒绝成为他人赚钱的机器和被剥削的奴隶。”
  <br/>
  <br/>
  我和许多反对派人士看到，躺平主义是权利意识和身份意识的觉醒，也从中看到了不合作思潮。所以，“躺平是觉醒，更是不合作，觉醒、不合作，中国就有希望”。
  <br/>
  <br/>
  网上有一首诗是这么写的：站不起来，但又不想跪着，就只有躺平。躺平，是为了不弯腰；躺平，是为了不下跪；躺平，是横向的站立；躺平，是挺直的脊梁。这首诗形象地表示了普通人面对残酷内卷的挥洒态度，也形象表达了反对派人士面对暴政的不合作姿态。
  <br/>
  <br/>
  “躺平主义”让当局恐慌，官媒和御用文人纷纷表态，指责躺平可耻、不负责任，是毒鸡汤…...。豆瓣上那个“躺平小组”被炸了，页面不存在了，还留着一行小组简介：躺平是一门哲学，躺平是一种艺术。在这个喧嚣的时代，以不变应万变……，躺平小组被炸了，躺平者四散了，向广袤的世界蔓延。网络上有一段广泛流传的视频，一位躺着的青年女子在说：“年轻人躺平到底惹了谁，我们这一代年轻人生活有多倒霉!”“年轻人不缺奋斗的决心和信心，年轻人缺的是奋斗路上的那束光。”
  <br/>
  <br/>
  因此在我看来，消极抵抗的躺平，不合作的躺平，正酝酿着直接反抗的情绪和能量。当那束启示中国人奋斗的光，在中国上空闪亮的时候，躺平挺直的脊梁就会霍然站立，专制暴政的末日就要到了！
