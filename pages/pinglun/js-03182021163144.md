 <p>
  中共人大、政协“两会”模仿外国众议院、参议院模型，给外界的印象是中国的“国会”。然而众所周知，在一党专政下，全国人大实为橡皮图章，全国政协沦为政治花瓶，都是拍手、举手的政治表演工具。
  <br/>
  <br/>
  但习近平犹嫌不够，在他上台后，一步步地又把两会变成了习家军权力斗争的工具。在2021年的两会上，这种工具色彩更形明显。
  <br/>
  <br/>
  今年的全国人大，在习家军人物栗战书、王晨等人的操纵下，通过了所谓“修改全国人大《组织法》的决定”。其中，最引人注目的是增加的如下两条：
  <br/>
  <br/>
  第三十一条：“常务委员会在全国人民代表大会闭会期间，根据国务院总理的提名，可以决定国务院其他组成人员的任免；根据中央军事委员会主席的提名，可以决定中央军事委员会其他组成人员的任免。”第三十二条：“常务委员会在全国人民代表大会闭会期间，根据委员长会议、国务院总理的提请，可以决定撤销国务院其他个别组成人员的职务；根据中央军事委员会主席的提请，可以决定撤销中央军事委员会其他个别组成人员的职务。”
  <br/>
  <br/>
  综合推敲这两条，发现两个微妙：其一，人大常委会扩权，可以在全国人大闭会期间任免或撤销国务院和中央军委的重要成员，显示对近三千人大代表不信任，把权力收缩到关键少数，即由170人组成的人大常委会。其二，人大闭会期间，有关中央军委的官员撤销，仍由军委主席提请；但有关国务院的官员撤销，既可由国务院总理提请也可由委员长会议提请，等于说，撤销官员，委员长会议并不越过军委主席，却可以越过国务院总理。
  <br/>
  <br/>
  考虑到现任人大委员长、军委主席、国务院总理分别是栗战书、习近平、李克强，而栗战书又是习近平的亲信、心腹、排头的习家军人物，扩大人大常委会的权力，就等于扩大栗战书的权力；越过李克强而不越过习近平，就等于习近平和栗战书联手对付李克强。这正反映中共高层的权力斗争格局：习家军对垒团派。
  <br/>
  <br/>
  可见，这次修改全国人大组织法不是出于法制建设，也不是着眼于长治久安，而是出于权力格局，着眼于明年二十大之前的权力博弈；并非长远战略，而是短期策略，服务于习近平和习家军的急功近利。
  <br/>
  <br/>
  修法事关重大，这次修改全国人大《组织法》是近四十年的第一次，然而并非因事修法，而是因人修法。仅仅因为习近平需要僭越任期、实现明年连任，急欲随时调配国务院和中央军委的人事，就擅自修改人大法规。正如2018年的两会，仅仅因为习近平个人追求长期执政或终生执政，就擅自和轻率修改宪法、废除国家主席任期制。
  <br/>
  <br/>
  这一切，足见在习近平治下，中国法制倒退、沦丧到何等程度！且不说绝无向前推进民主政治之意，就连中共自己改革开放的些微政治进步，都被习近平和习家军剥夺殆尽，而走向全面的政治倒退，从一党专政升级为一人专制，从集体领导倒退为一人独裁。
  <br/>
  <br/>
  这固然是国家和民族的悲哀，却也是作为执政党的中国共产党的悲哀。难以想象，一个九千万人的大党竟然无力阻止一个人的恣意妄为！一切都是制度使然，一党专政成为一人独裁的工具，一人独裁又成为一党专政的死结。一党专政压迫这个民族，一人独裁又压迫这个党，双重压迫叠加，恶性循环。长此以往，最后的结局只能是同归于尽。
  <br/>
  <br/>
