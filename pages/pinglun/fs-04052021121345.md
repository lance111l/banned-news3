 <p>
  二战结束后，美国向被战争蹂躏的国家伸出援手，通过马歇尔计划帮助他们重建经济和社会。几十年后，美国用同样模式帮助中国实现了现代化，并相信随著时间的推移，中共也会开放其经济和社会。然而，美国和西方的愿望落空了。中共从自由开放的国际秩序中攫取好处，却处心积虑要毁灭这个秩序，构建红色帝国的新秩序。做着红色帝国春梦的习大帝在香港、台湾、新疆，以及疫情等各方面展露的咄咄逼人的战狼嘴脸，促使了西方的觉醒，开启了西方对中共的重新认识。
  <br/>
  <br/>
  总部设在加拿大哈利法克斯的哈利法克斯国际安全论坛，发布了一份题为《中国与民主国家—世纪最大博弈》的手册。该论坛主席彼得·范·普拉格（Peter Van Praagh）在报告的前言中这样写道：“ 2020年，见证了民主世界对中国共产党的态度发生了根本性的转变。具体表现为：如果说，过去传统的看法是经济繁荣的中国最终会为其民众带来更多的自由，那么新的流行看法是，中共才是危及全球的病毒。”
  <br/>
  <br/>
  在美国，两党越来越一致地认为，中共现在是民主世界及其价值观的主要挑战者。美国坚定的传统盟友加拿大和英国，也出现了同样的觉醒的迹象；欧盟也有类似的共识；印度目前正在采取重大措施， 重新调整对其东部邻国的政策；日本一直对中国忧心忡忡。
  <br/>
  <br/>
  西方觉醒的表现还有，中国科技公司在西方越来越被排斥。越来越多的国家宣布，将华为5G排除在自己的电信系统建设之外；孔子学院，以及大外宣在越来越多的国家遭遇抵制。
  <br/>
  <br/>
  尽管许多西方国家政客为了中国的市场和个人利益，继续和中共勾兑、采取绥靖政策，但认识到中共国是不承认西方价值观的大国，对西方形成致命的威胁，西方必须采取措施应对中共的挑战已经是大势所趋。最亲近中共的德国也开始转向。
  <br/>
  <br/>
  曾经一蹶不振的美、日、印、澳四方联盟重获启动。但前国家安全副顾问麦克法兰说：如果这仅仅是四国集体抱怨中共恶行，如果没有具体行动，只是靠言语和愿望，最终西方将处于下风。她建议，建立以“四国联盟”为开端的经济和安全联盟体来遏制中共的扩张，就有可能维护国际秩序而不受中共的霸凌。她说，如果美国现政府不愿意或不能从四国同盟开始打造一个安全联盟，那么下一任共和党总统将不得不在2024年动手。
  <br/>
  <br/>
