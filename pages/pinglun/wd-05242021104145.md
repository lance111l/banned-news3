 <p>
  <br/>
  最近几年，中国年轻世代中流行“小粉红”现象，而且愈演愈烈。不管外界有多少不解，但确实有为数不少的90后和00后世代对中共的统治是拥护的，对中国的现状是满意的。他们不分是非，把所有中国的问题都归咎于“境外敌对势力”，而且带有强烈的狭隘民族主义倾向。这样一代人的形成当然不是我们乐见的，也有碍于中国社会的文明进步，但一味的责备与不齿无助于事，我们还是要去理解和正确看待“小粉红”现象。
  <br/>
  <br/>
  首先，我们要理解小粉红现象的形成原因。这一代年轻人没有经历过“文革”，没有经历过“六四”，用现在流行的话说，他们“没有尝到过社会主义的铁拳”；相反，他们从出生到成长，看到的是中国经济增长带来的表面上的繁荣。由于他们的阅历尚浅，更由于中国言论自由的匮乏，他们无法看到繁荣表象之下的黑暗与代价，因此毫无保留地认为自己所处的社会是一个进步的社会，认为中共的治理是有成绩的。这在某种程度上，也是可以理解的。但是我认为，随着他们的成长，随着他们与社会的接触范围更加扩大，他们的认识也会逐渐发生变化。当初80后世代也曾经“粉红”过，但现在年过三十，整体性的“粉红度”已经大为下降。这样的变化，也会发生在现在的“小粉红”世代身上。
  <br/>
  <br/>
  其次，我们也一定要认识到，今天的小粉红现象，在很大程度上是中共当局在背后操纵和推波助澜的结果。在中国，民族主义情绪的起伏完全掌控在中共的手中，如果没有他们的允许和支持，没有任何民族主义情绪滋生和发泄的余地。长期以来，中共依赖经济增长和民族主义作为支撑其统治的基础。现在，经济增长这根支柱已经摇摇欲坠，中共只能，也必须，用更大的精力来维护和加强另一根支柱，也就是民族主义。近几年，中国“战狼式外交”大为流行，令外界震惊。但背后的逻辑结果是很清晰的，那就是经济增长停顿之后，民族主义对于中共的重要性的增加。在这样的背景下，中共也加强了对年轻世代“小粉红”现象的引导，其中，共青团等组织在网路上进行的各种宣传就是典型的例子。因此，“小粉红”现象并不简单地是一个中国年轻世代的特质的问题，它更折射出了中共统治面临的危机，以及他们采取的因应对策。“小粉红”现象的出现和强化，就是中共采取措施应对危机的结果。
  <br/>
  <br/>
  最后，我们也没有必要过分夸大中国年轻世代的“小粉红”现象。在没有言论自由的今日中国，年轻世代中“小粉红”的比例到底有多大？这些“小粉红”的影响力有多强？都是非常值得商榷的。他们声浪高，某种程度上是因为他们对于网络的熟练运用，使得他们的言行更加可以被看到。在真实的世界里，当他们逐渐走出家庭的庇护，开始承担一些生活的压力的时候，他们是否还能保持对中共的信心，也是令人怀疑的。而即使是那些最为积极的“小粉红”，也没有多少人会真的拒绝西方资本主义的物质消费。仅凭这一点而言，他们的“粉红”生涯恐怕也不会在他们的人生中伸展很远。换句话说，“小粉红”世代变化很快，中共要维持中国的“小粉红”大军，必须不断地、重复地洗脑新的世代，这并不是一件容易的事情。
