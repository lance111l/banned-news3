<div id="storytext">
 <p>
  近日美国媒体报道，中国新建了多达145个核导弹发射井，其中仅在甘肃玉门附近的沙漠中就有119个。这可不像是空穴来风，有近期卫星照片为证。还有专家分析，它们部署的是中国最先进的东风-41洲际弹道导弹。
  <br/>
  <br/>
  这类消息的最特殊之处在于，美、中两国官方的惯例都是不会评论其事实的真实性。而外界，包括国际军控研究界，除了能发现这种现象，也不具备进一步准确核实它们是否具备洲际导弹发射井的能力，只能在“宁可信其有”的前提下沸沸扬扬。
  <br/>
  <br/>
  外界或许认为，眼见为实。荒野上这一大堆巨型混凝土井，的确没有其他合理用途。中国没有那么多导弹，建这么多发射井做什么？反之，已经有这么多井，难道不必然说明中国已经导弹大跃进了吗？
  <br/>
  <br/>
  但是，如果就此认定中国陆基核导弹已经翻番，恐怕正中中国下怀。
  <br/>
  <br/>
  在核战略中，密集部署核导弹的想法并不新鲜，却非常恐怖。它意在通过适当紧凑地布置发射井，使敌方企图用大量核弹头一举荡平己方发射井时，既不会一发摧毁两个以上。但其众多核弹头在短时间内同时起爆，杀伤效果相互削弱。
  <br/>
  <br/>
  类似的陆基核导弹部署方式还有加固井、超深井、空中/公路/铁路机动、隧道群机动和水下漂浮等，都是美苏核军备竞赛顶峰时期动辄设想全面核战争的产物。因为美苏这样的国家也难以负担这些部署方式的耗资，最终只有加固井和公路机动被普遍采用，并与海基和空基核力量共同构成可靠的威慑力。
  <br/>
  <br/>
  <strong>
   核导弹发射井模式落后 皆因容易遭到打击
  </strong>
  <br/>
  <br/>
  中国立足于小规模、自卫反击的少量核力量也需要避免被对手抢先消灭，除少量发射井外，主要依靠加固坑道和公路机动的手段，成本尚属适度。这些数量有限的核导弹在全中国公路范围内不停移动，已经令美国无法抢先一次性消灭，集中部署到发射井反而树大招风。
  <br/>
  <br/>
  中国虽然因为美国导弹防御系统能有效削弱核反击规模，而多次发出大幅增加核弹头数量的威胁，但财力负担和军备竞赛压力一直是难题。因而，中国核导弹因反导压力而增加数量，但与它们因面临被抢先摧毁的压力而采用特殊部署方式，是核对抗之中，也是核力量发展的两个不同方向的课题。为应对反导而适当增加数量，还是立足于自卫反击。而采用密集发射井部署既意味着导弹数量成倍猛增，与美国展开全面军备竞赛，还意味着核对抗升级。在现有百枚基础上翻倍当然足以压倒美国的导弹防御，但要与美国达到数量均势尚差起码6倍，而美国的应对完全可以不限于反导。美国要恢复核武库比中国轻松得多，还会使经济衰败的俄罗斯暗自叫苦。
  <br/>
  <br/>
  技术上的一些疑点尚在其次。比如这次建设的发射井间距3公里左右，远大于美国当年反复研究得出的500余米的最佳井间距，难以用美国历史案例来理解。全国范围145个发射井若119个集中一地，其余26个又如何生存？有何意义？最容易被质疑的东风-41等导弹没有井基型反而不是问题，需要有就很容易有。
  <br/>
  <br/>
  只要这项工程真的存在，北京应该也完全知道它不仅躲不开美国军方，近年连国际社会的视线也越来越躲不开。建设之初，难道中国就意识不到此举会造成疯狂核扩军形象，在美俄均有意将中国纳入核裁军进程的局面下，对自己极为不利吗？
  <br/>
  <br/>
  <strong>
   虚张声势策略只会招致国际社会提高对中国的警觉性
  </strong>
 </p>
 <p>

  <br/>
  对此，可以参照中国在南沙群岛耗资巨大的世界级吹沙填岛工程。该工程虽然的确可以部署战斗机和导弹，但引来的国际压力之大，对周边关系破坏之深，实际上又使中国武力解决南海问题的军事难度并无根本改观。尽管得到的只是一个效果不佳的僵局，但专家成群的中共就是没有预见，偷鸡不成蚀把米，丝毫不妨碍中国一言不和就满世界灌水泥的信心。毕竟灌水泥是中国的看家本领，比造洲际导弹容易、廉价得多。
  <br/>
  <br/>
  在核军备领域，外界长期无视一个现实：核扩军对中国也是费劲又烧钱。同放弃“韬光养晦”，自不量力地在全球四面出击一样，中国超越既定的“精干”原则，在核军备方面“奋进有为”也是完全符合习近平个性的一种真实冲动。但再自欺欺人，中共高层也不可能不知道，自己完全没有实力与美国在数量、质量上展开双重竞赛。因此，中国官方对胡锡进等人“增加一千枚核武器”的鲁莽叫嚣不置可否，在行动上也面临现实制约，表面高歌猛进，其实颇为迷乱，不无难处。
  <br/>
  <br/>
  对裁军和舆论压力，中共反而自有一套。贸易战那样实质利益损失巨大的领域尚且“敢于斗争”，裁军就是不参加，你奈我何。以往对“中国威胁论”的不安，不过是“韬光养晦”的配套，现在中共虽然口头上还想“讲好中国故事”，但对抗时代已至，“斗争”才是主题。
  <br/>
  <br/>
  一旦抛开顾虑，中共近几十年从遮遮掩掩到大张旗鼓的强（扩）军过程，还产生了一个额外的政治红利：中共军备方面的任何动向，哪怕完全是虚张声势或者纯属误解，也会被西方过高估计。西方的“惊呼”都会被中国喉舌作为激发爱国热情和强国自豪的特效春药，使中共收割巨大的政治支持率。每当外国记者就这种消息求证，中国军方一面回避正面回答，一面强调中国强军决心和正当性的态度，总能巧妙地勾起外界更加火爆的猜想。
  <br/>
  <br/>
  如同西方对朝鲜的核坑道和伊朗的离心机不断猜测，对两国实际的核武规模、水平和潜力却永远不确定一样，“你猜，你使劲猜”，已成部分国家挑战（如果不算讹诈的话）美国的得力武器。即使是西方军控、学术界，因为以实证和量化分析见长，对苏联和中国某些军力的估计在历史上就往往偏高。
  <br/>
  <br/>
  <strong>
   中国核威胁有助煽动国内仇外和军国主义情绪
  </strong>
  <br/>
  <br/>
  看见百多个水泥井就认定中国核武库翻番，西方当然可以在军控、外交和舆论领域对中国大加抨击，可中共早说了：“不吃你那一套”。无法理解，中共何以敢重蹈苏联军备竞赛的覆辙吗？你能查证这些地下工程的真正质量和完成度吗？你看见导弹了吗？中国难道不能只将其中部分发射井真正完成并部署真导弹，从而以合理成本收获虚张声势和狡兔三窟之效吗？中国离裁军还差十万八千里，你还能现场核查不成？
  <br/>
  <br/>
  不管中国是否下决心展开军备竞赛，这方面的信息越耸人听闻，越一团乱麻，或许也越为中国高层喜闻乐见。西方即使群起而攻之，矢口否认即可，但是保持旺盛的仇外敌意和军国主义情绪，渲染西方阴谋和对抗态势，吹嘘强军进展和敢打必胜勇气，却是中国国内形势最急需的政治资源。与之相比，在核军备上野心过大，战略混乱，具体工程上考虑不周，效果不佳，被抓住些许把柄引来“友邦惊诧”，都不算什么。
