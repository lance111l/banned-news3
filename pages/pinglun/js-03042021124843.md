 <p>
  近期，中共宣布中国全面脱贫，达到全面小康的政治目标。习近平自夸：这个全面脱贫是“千年级别的大成就” 。果真如此吗？
  <br/>
  <br/>
  先不说千年，且说中共改革开放之后的四十多年，脱贫、小康这些词汇在不同阶段、不同历史条件下，或有不同标准却有相同意涵。
  <br/>
  <br/>
  早在1978年，分别担任四川省和安徽省第一把手的赵紫阳和万里，在这两省尝试“农村联产承包责任制”，松绑了两省农民的手脚，焕发了他们的积极性，使得这两省农民迅速获得温饱，以至于有“要吃粮，找紫阳；要吃米，找万里”的美谈。四川和安徽的经验随后推广到全国，仅在两年内，到1980年，就基本解决了全国农民的吃饭问题，从饥饿到温饱。按当时的历史背景，那就是毛泽东饥饿时代之后的首次脱贫，也可称为当时的全面脱贫。
  <br/>
  <br/>
  在此基础上，后来中国的国民生产总值GDP又实现翻两番，又算是另一个阶段的脱贫；而在胡温时代，GDP甚至连上五个十万亿台阶，并首次取消农业税，更称得上当时条件下的全面脱贫。紧接着胡温上任的习近平，竟然贪天之功，把所有脱贫、小康的功劳全摊在自己头上，宣称在他的任内才实现了“全面脱贫、全面小康”，而一概抹杀他之前的所有前任。
 </p>
 <p>
  <img alt="1" class="image-richtext image-inline captioned" src="https://www.rfa.org/mandarin/pinglun/chenpokong/js-03042021124843.html/f65c06a8-90de-47dd-86c9-1a0029e5072c.jpeg" title="1"/>
  <br/>
  <br/>
  再说千年。在上千年或者数千年王朝循环的中国历史上，几乎每个王朝都有其大治、中兴、鼎盛时代，称为盛世。其基本特征都是，民间丰衣足食、国家太平安康。中国人耳熟能详的就包括周朝的“成康之治”、汉朝的“文景之治”、唐朝的“贞观之治”和“开元之治”、清朝的“康乾盛世”，等等。往小的说，都是小康社会；往大的说，则是民富国强。
  <br/>
  <br/>
  由弱而强，由盛而衰，这是历史上中国王朝的兴衰败亡规律。继承了历朝衣钵的新专制王朝 -- 共产党红朝，不过就是一个朝代的循环再现，重复兴衰败亡的规律。当今红朝或正处于它的“盛世”，但因其专制政治的本质并无变革，其由盛而衰的命运必将不期而至。
  <br/>
  <br/>
  由此推之，眼下由习近平当局自行宣布的“全面脱贫、全面小康”，不仅是自说自话，而且绝不是所谓“千年级别的大成就”。
  <br/>
  <br/>
  况且，要说“千年级别的大成就”必经得起千年的检验。比如，位于成都平原的都江堰于公元前256-251年的战国时期兴建，兼有灌溉、防洪、供水、运输之用，历两千三百年而不倒，惠泽天下至今，这才堪称“千年级别的大成就”。其卓越功勋和历史定位，并非由当时主建这一水利工程的李冰父子定义，而由后世近两千三百年的口碑铸就。
  <br/>
  <br/>
  习近平何德何能？居然自我定义、自我吹嘘他任内推动的脱贫工作是“千年级别的大成就”。且不说国际上是否认同，中国国内就不见得认同。而在中国国内，且不说民间服不服，就说党内就很可能不服。政治高压下的表态，也只是口服心不服。
 </p>
