  退出共产党声明

 <br/>
 我是陈慧。在这里，我郑重声明退出中国共产党、共青团、少先队等与中国共产党相关的一切组织。
 <br/>
 我长期以来一直受到各种压迫和限制，我的工作和生活受到了领导的各种作威作福和操控，我的思想和信仰受到了极大的干扰和扭曲。我逐渐看到了共产党的本质，看到了他们残忍无情的迫害、镇压和控制，看到了他们对人民生命权、自由权、财产权的肆意践踏和摧残，看到了他们肆意欺压和摧毁人民的尊严和人权，看到了他们对社会的摧毁和对文化的扭曲，看到了他们对国家的危害和对世界的负面影响。
 <br/>
 我曾经是预备党员，但在点头哈腰求一个党员的途中，我看到了共产党邪恶的本质。我不愿再为这个邪恶的组织效劳，我也不再想被他们控制和操纵，我也不想我的家人、我的儿女以后被共产党控制。今天，我宣布与这个邪恶组织永别，彻底割断与它们的联系，重新拥抱自由、真相和正义，重新拥抱我的信仰和良知。我将永远支持民主、自由、人权和法治，为一个更加美好的未来而努力奋斗。
 <br/>
 最后，我希望所有被压迫和迫害的人民都能够勇敢地站出来，加入到抵制邪恶的行动中来，共同为一个更加美好的未来而努力奋斗。
 <br/>
 声明人: 陈慧

<p>
 <strong>
  <ok href="https://www.epochtimes.com/gb/tag/%E9%80%80%E5%85%9A.html">
   退党
  </ok>
  、退团、退队声明
 </strong>
 <br/>
 真正的自由是发自内心的相信真正的自己，容纳一切真善美。
 <br/>
 而今来世已几十年有余，从小在耳濡目染之下听信了中共的一面之辞，在不自知的情况下与中共同流合污，不久前才渐渐醒悟，知道了只信一面之辞不过迷信尔，看待问题必须从多方面进行探索并实证，如此才不至于被他人蒙骗，成为恶徒的帮凶。
 <br/>
 而当今中共的所做所为已逐步被普众所知，身为普通人的我们能切身地感受到中共的谎言与欺骗：口口声声说人人平等，却将人分为三六九等（连农民证这种东西都整出来了）；口口声声说为人民服务，做人民的公仆，而现实却是人民为你们服务，做你们的公仆，真到了人民需要你们的时候，就是一个字“拖”（拖拉机都没你们能拖）；口口声声强调意识形态，实际上却是四处树敌，最后暴力解决，任意杀戮，甚至歪曲掩盖事实真相，你们真的有尊重生命的想法吗？还是说只是口头说辞？（而今倒好，你们与自己口中的敌人“资本”达成了共识，共同压迫人民。当然，资本也不是什么好鸟，其目的无非是操控人，然后将“名利欲知”等集中在少数人手中，这几千年以来的“家天下”轮回，究竟何时结束？此刻方知天下的乌鸦一般黑，人们能相信的只有自己的良心）。
 <br/>
 此时此刻人民敢怒不敢言，正如《阿房宫赋》所说的“独夫之心，日益骄固”，统治者及其走狗的最终下场，不过是“楚人一炬，可怜焦土”。为什么人们总是不能从历史中吸取教训？不能深刻地认识到自己是谁？不能清楚地认识到自己该做什么？甚至喜欢聚成一个团体去打压、攻击、抹黑其它团体，由此而形成了恶性循环的“斗争”。世人只知道敌人的恶毒，而不知道自己的恶毒，因此时常以恶为善，不辨真假，肆意发泄心中的怨恨与不满（当然，这句话是我以“道”的视角来阐述的，我本人没有资格说出这种话）。
 <br/>
 在此之上，我逐步明白了：真正的逍遥自由是不受万事万物约束而凭良心存活；真正的道理从来都在“目前”，而不在外物；真正的世界从来都不是他人的一面之辞，而是自己内在的觉知、本性与良心。以上是我对“道”的个人理解，具备人自身片面的思想，真正的道是“只可意会不可言传”的，每个人选择的道路都是不同的，没必要强制逼迫他人去服从你的“道”。因此，我的话只能当做一个参考，不可以执一以贯之。此刻你需要知道并选择自己的道路！这是人生而为人的选择。
 <br/>
 从今天起，你不再需要执著，不再需要沉迷，不再需要欺骗，当下的每一刻都是最好的自己，学会容纳万物、承接万物、运化万物、使万物与我合一，我是自己的，也是整个世界的，我是存在的，也是不存在的，是与不是，是由我选择的，心与物原来竟一体的！
 <br/>
 故当下的我选择退出的任何中共组织，不愿做任何人、事、物的“奴隶”。以上的文字表明了我的选择，同时我也明白了“一阴一阳之谓道”的奥妙，如若不是长期地被中共、资本等团体及其自己所欺骗，我也不会有现在的觉悟与反省（无产与资产相互敌对，国外与国内相互隐瞒真相，普众难分真假善恶，只能不知所谓苟且而活，当今只有凭良心而致公心大同），我不再会排斥任何事物，我会容纳一切合理，我不会再执著于好坏善恶，我将用心去改变自己与当下，此刻的我尚且无法做到全部，所以我没有资格批判任何人或任何组织，但长期被欺骗的感觉真的很不舒服，所以我写了这篇文章，发泄了我心中的不满，这种行为本质上是错误的！因此我做的还不够，我的心还不够静，神还不够清，所以见我文章者不要听信我的一面之辞，要去自己寻找真正的自己，保持一颗向善的心，并慢慢去寻找真实的自己。
 <br/>
 以上是我退共产党、退共青团、退少先队的
 <ok href="https://www.epochtimes.com/gb/tag/%E4%B8%89%E9%80%80.html">
  三退
 </ok>
 声明，同时也是我自己的忏悔文，我并不是为谁而忏悔，也不是为了向世人说明什么，我只知道我是属于我自己的，也是属于这个世界的！在此我也清楚中共组织里并不全是恶徒，也有真心为民的好官员，一棒子打死是不可能的，但是非分明也很难。善恶可以异形，好坏可以相换，本来无方位，何来进退之说？只愿世人能相信自己、相信一切，安稳度过灾劫，如此就够了。
