 【大纪元2023年02月08日讯】（大纪元记者林燕综合报导）
 <span class="s1">
  美国参议院多数党领袖查克‧
  <span class="s2">
   舒默
  </span>
  （Chuck Schumer）在上周日（2月5日）召开的记者会上透露，在公众知晓气球的存在后，北京曾试图操控监控气球使其尽快离开美国。
 </span>
</p>
<p class="p1">
 <span class="s1">
  这是外界关注的一个焦点。因为中共外交部一直称，气球是失控、意外偶发事件。
 </span>
 <span class="s1">
  舒默的说法直接打脸中共的气球说辞。
 </span>
</p>
<p class="p1">
 <span class="s1">
  “我们确实知道，当气球在公众面前曝光后，中方曾试图操控气球（使其）尽快离开美国。”他说。
 </span>
</p>
<p class="p1">
 <span class="s1">
  舒默还说，由于该监视气球是在美国领海上空击落的，美国有能力回收大部分碎片，“我们可以对中国（中共）的能力和他们的所作所为了解很多”。
 </span>
</p>
<p class="p1">
 <span class="s1">
  他已经安排在2月15日为
  <span class="s3">
   全体
  </span>
  参议员举行跟
  <ok href="https://www.epochtimes.com/gb/tag/%E4%B8%AD%E5%85%B1%E7%9B%91%E6%8E%A7%E6%B0%94%E7%90%83.html">
   中共监控气球
  </ok>
  有关的简报会。国防部理论评估办公室（Office of Net Assessment）原本计划在当天就中国（中共）的军事实力和监视基础设施提供简报，气球事件的简报将一并进行。
 </span>
</p>
<p class="p3">
 <span class="s1">
  舒默说，简报将包括有关中共的监视能力、研究和开发、先进武器系统和其它“关键平台”的信息。
 </span>
</p>
<p class="p1">
 <span class="s1">
  上周
  <ok href="https://www.epochtimes.com/gb/tag/%E4%B8%AD%E5%85%B1%E7%9B%91%E6%8E%A7%E6%B0%94%E7%90%83.html">
   中共监控气球
  </ok>
  出现在美国上空在华盛顿引起一场激烈的政治反应，并促使国务卿安东尼‧布林肯（Antony Blinken）取消了周日至周一对北京的访问，中美两国领导人之前都希望通过外交来缓和变冷的双边关系。
 </span>
</p>

<h4 class="p11">
 舒默：拜登正在研究可以采取的其它行动
</h4>
<p class="p1">
 <span class="s1">
  舒默周二（7日）在国会再次告诉媒体，美国与中国的关系现在很紧张，拜登政府正在研究下一步可以采取的其它行动。
 </span>
</p>
<p class="p1">
 <span class="s1">
  “我知道白宫正在研究可以采取的其它行动。”舒默说。
 </span>
</p>
<p class="p1">
 <span class="s1">
  民主党总统拜登准备周二晚在美国国会发表年度国情咨文，因为在应对中共间谍气球事件上的明显拖延，他一直被共和党人批评。
 </span>
</p>
<p class="p1">
 <span class="s1">
  舒默试图缓和这一批评，他呼吁，两党要共同谴责中共的所作所为。
 </span>
</p>
<p class="p1">
 <span class="s1">
  “这是一个不需要政治的领域。所以，我们需要民主党人和共和党人走到一起。我们需要国家团结起来，谴责中国（中共）的所作所为，并在处理中共问题时有一个统一的阵线。”他说。
 </span>
</p>
<p class="p1">
 <span class="s1">
  拜登本人在周一表示，美中关系没有因为气球被击落而被削弱。“不，我们向中方明确说明了我们要做什么。他们理解我们的立场。我们不打算退缩。”
 </span>
</p>
<h4 class="p1">
 外国记者接连追问 中共外交部招架不住
</h4>
<p class="p1">
 <span class="s1">
  在周一（6日）的中共外交部例行记者会上，美国记者追问发言人毛宁，能否确认舒默所说。毛宁拒绝直接回答提问，反而强调中方
 </span>
 <span class="s4">
  一直“负责任”以及美方反应过度。
 </span>
</p>
<p class="p5">
 <span class="s5">
  整场记者会都是不同外国记者就中共监控气球的不同提问，但中共发言人给出的答案都是同一个。
 </span>
</p>
<p class="p5">
 <span class="s5">
  比如，针对中方的矛盾立场，一会儿抗议击落，一会儿捍卫权利，记者提问，“中共的立场究竟是保留击落未经授权进入领空的无人飞行器的权利，还是认为不应击落未经授权进入一国领空的无人飞行器？”
 </span>
</p>
<p class="p7">
 <span class="s5">
  其次，“中方是否意识到当前局面有多危险，是否应避免针锋相对引发局势升级，尤其是如果美方搜集到的碎片显示气球确实是间谍气球的话，怎么办？”
 </span>
</p>
<p class="p7">
 <span class="s5">
  还有记者就中方气球缺乏全面系统的追踪能力提问，“如果只能依靠其它国家通报
  <ok href="https://www.epochtimes.com/gb/tag/%E4%B8%AD%E5%9B%BD%E6%B0%94%E7%90%83.html">
   中国气球
  </ok>
  进入他国领空，中方对此是否感到担心？”
 </span>
</p>
<p class="p9">
 <span class="s5">
  “中方一再称，误入美国和拉美上空的气球都是意外偏离路线，请问中方为何很难控制自己的气象气球？”
 </span>
</p>
<p class="p7">
 <span class="s5">
  “中共外交部是什么时候了解到气球的有关情况的？是在美方宣布发现气球之前还是之后？”
 </span>
</p>
<p class="p11">
 <span class="s1">
  针对诸多不同方面的问题，毛宁都套用同一个说词回应，
 </span>
 <span class="s6">
  同时警告美国“不要没完没了”，并称“
 </span>
 <span class="s1">
  中方将视事态的发展作出必要的反应”。
 </span>
</p>
