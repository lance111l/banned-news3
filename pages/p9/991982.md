   朝鲜金正恩

  当局为了防范派至海外的工人出现
  <strong>
   脱北
  </strong>
  行为，而加强监视与管控力度，甚至还采取残酷之措施。据悉，由于
  <strong>
   边境封锁
  </strong>
  的长期化，使得
  <strong>
   海外劳工
  </strong>
  滞留时间变长，也让平壤当局非常忧心工人们会引发脱北事件。
  <br>
   <br>
    8日，根据DailyNK俄罗斯消息人士透露，在4日的政治学习时间中，下达了报告工人动向与加强管控的指示。
    <br>
     <br>
      据了解，俄罗斯当地工作站站长、党书记、负责人及保卫指导员等管理工人的干部们参加了政治会议。
      <br>
       <br>
        据悉，暂时还不会交替人员，签证到期者及已满最长期限10年的人员，将会再给延长1年的滞留时间。
        <br>
         <br>
          学习上更强调称，这个措施并不是因为对于
          <span href="https://zh.wikipedia.org/wiki/%E6%9C%9D%E9%B2%9C%E6%B0%91%E4%B8%BB%E4%B8%BB%E4%B9%89%E4%BA%BA%E6%B0%91%E5%85%B1%E5%92%8C%E5%9B%BD">
           朝鲜
          </span>
          的制裁，而是依据防范传染病流入国内防疫措施所决定的。而是因为目前的Omicron病毒株逐渐扩散，因此担忧滞留海外的人员回国后，将可能会瞬间弄垮国内防疫壁垒。
          <span id="hideid" name="hideid" style="color:red;display:none;">
           <span href="https://www.secretchina.com">
           </span>
          </span>
         </br>
        </br>
       </br>

   </br>

 <p>
  朝鲜
  <span href="https://www.secretchina.com/news/gb/tag/金正恩" target="_blank">
   金正恩
  </span>
  当局为了防范工人在海外发生脱北事件，而祭出了更加强硬的管控政策。当局于公告中声称，必须彻底监视平时怨言多、举动有可疑及想背叛祖国逃跑之人员，且随时报告情况，尤其应注意发生违法行为。如果背叛祖国逃跑了，将尽所有手段来展开逮捕，并依据国家保卫省之遣送程式来遣送回国。
 </p>
 <p>
  在俄罗斯的海外劳工们皆知道国家保卫省所规定的遣送程式，也就是弄残试图脱北者的腿，而令其回国时无法自行走路。
  <br>
   <br/>
   曾经至俄罗斯当过劳工的
   <span href="https://zh.wikipedia.org/wiki/%E8%84%AB%E5%8C%97%E8%80%85">
    脱北
   </span>
   者A披露，其实，劳务人员都清楚国家保卫省内部所规定的遣送程式，亦即弄断腿让他无法走路，而用轮椅遣送回国。但尚未见过让人很惊恐的国家亲口所说的按程式遣送回国。
  </br>
 </p>
 <p>
  据朝鲜内部高层消息人士表示，派至俄罗斯当劳工的朱京哲决定想逃到韩国之后，他在2017年1月并末至驻俄罗斯联合国办事处来寻求帮助。在朝鲜国家保卫省逮捕组前往俄罗斯逮捕了朱京哲后，就先挑断其足筋，并进行麻醉后遣送回朝鲜的。他遭遣送回朝时的年龄为28岁。
  <br/>
  <br/>
  国家残忍地伤害个人身体才进行遣送，由此可见
  <span href="https://zh.wikipedia.org/wiki/%E9%87%91%E6%AD%A3%E6%81%A9">
   金正恩
  </span>
  政府很担心海外劳工的脱北行为。
 </p>
 <p>
  <span href="https://www.secretchina.com/news/gb/tag/朝鲜" target="_blank">
   朝鲜
  </span>
  当局在这次的公告中也明确阐明，工人必须上交至国家的国家计划任务，也就是“忠诚资金”额度并没有变化。因COVID-19疫情，而无法替换海外劳工之情况下，朝鲜政府试图通过赚外汇来确保党资金。
  <br/>
  <br/>
  目前，派至俄罗斯的朝鲜工人，他们主要从事伐木、建设等工作，人均每年必须向国家交付4500美元至1万美元的党资金。
  <br/>
  <br/>
  据DailyNK采访得知，朝鲜政府于6月份提高了派往俄罗斯劳工的党资金额度，其涨幅高达30至55%。
  <br/>
  <br/>
  据悉，派往俄罗斯的工人一般都从事高强度工作，且生活环境十分恶劣，而让他们感到非常痛苦。在此种情况下上交的党资金又暴涨，使得很多工人希望能够回国。
  <br/>
  <br/>
  朝鲜政府阐明了不替换人员的同时，又要求继续上交上涨的党资金，因此预计派至俄罗斯的工人，其不满情绪将会高涨。但由于当局加强了监视力度，他们不可能会轻易决心脱北。
